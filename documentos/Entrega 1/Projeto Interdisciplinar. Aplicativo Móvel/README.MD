## 📖 Descrição do Trabalho  
Esta entrega refere-se ao desenvolvimento do **Projeto Interdisciplinar: Maya RPG**, com foco na implementação funcional do sistema.  

O trabalho consiste na construção do **servidor utilizando Node.js**, integração com **banco de dados na nuvem** e desenvolvimento do **aplicativo mobile no Android Studio**, garantindo a comunicação entre as camadas do sistema.  

O objetivo desta etapa é validar o funcionamento completo da aplicação, assegurando que o app esteja conectado ao backend e executando corretamente sua funcionalidade principal.

---

## 🎯 Objetivo  
Implementar o sistema principal do projeto, garantindo:  

- Comunicação entre aplicativo mobile e servidor  
- Execução correta da funcionalidade principal  
- Integração com banco de dados na nuvem  
- Correção de erros antes da entrega final  

---

## ⚙️ Tecnologias Utilizadas  

- 📱 **Android Studio** – Desenvolvimento do aplicativo mobile  
- 🌐 **Node.js** – Desenvolvimento do servidor backend  
- ☁️ **Banco de dados na nuvem** – Armazenamento das informações  
- 🔗 **API REST** – Comunicação entre app e servidor  

---

## 🔗 Funcionalidades Implementadas  

- 🔐 Autenticação de usuários (login e cadastro)  
- 📡 Comunicação com API backend  
- 💾 Persistência de dados na nuvem  
- 📱 Execução da funcionalidade principal do app Maya RPG
